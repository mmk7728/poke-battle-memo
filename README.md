# poke-battle-memo

ポケモン戦闘用の6匹パーティメモアプリケーション。パーティの詳細情報を記録・保存・エクスポート・インポートできます。

## 🎯 主な機能

### 📝 6匹のポケモン登録
- 各ポケモンに以下の項目を入力可能：
  - **ポケモン名** - 種族名
  - **特性** - ポケモンの特性
  - **持ち物** - 装備アイテム
  - **技4つ** - 4つの技を個別に登録
  - **テラスタイプ** - 18タイプから選択可能
  - **その他** - 役割や対策メモを自由に記入
  - **選出有無** - チェックボックスで選出メンバーを管理

### 🎯 アコーディオン表示
- ポケモン名のヘッダーをクリックすると、詳細情報の展開・折りたたみが可能
- スッキリとした見た目でパーティを管理できます

### 💾 パーティ管理
- **パーティ保存** - あらかじめ入力してあるパーティ名で保存
- **パーティ読込** - 保存されたパーティを復元
- **パーティ削除** - 個別削除・一括削除
- **フォームリセット** - 現在の編集内容をクリア（保存後は自動）

### 📊 CSVエクスポート・インポート
- **CSVダウンロード** - 保存されたすべてのパーティをCSV形式で一括ダウンロード
  - パーティ名、ポケモン情報、選出有無などすべてのデータを保存
  - 日付付きファイル名で複数回ダウンロード可能
  
- **CSVアップロード** - ダウンロードしたCSVを読み込んで復元
  - 複数のパーティを一括インポート可能
  - パーティの世代を超えた保存・移行に便利

## 📖 使い方

### パーティを作成・編集
1. `index.html` をブラウザで開く
2. 6つのカードのヘッダーをクリックして詳細を展開
3. ポケモンの情報を入力
4. 選出メンバーにするなら「選出メンバー」のチェックボックスにチェック

### パーティを保存
1. パーティ名入力欄に保存したいパーティ名を入力
2. 「パーティを保存」をクリック
3. 自動で内容がリセットされ、次のパーティの作成へ

### パーティを読み込み
- 保存されたパーティ一覧から「読込」ボタンをクリック
- フォームに前回の内容が復元されます

### すべてのパーティをバックアップ
1. 「CSVダウンロード」ボタンをクリック
2. 日付付きのCSVファイルがダウンロード
3. 別のデバイスやブラウザで同じバージョンを使用する場合に復元可能

### CSVからパーティをインポート
1. 「CSVアップロード」ボタンをクリック
2. 前に保存したCSVファイルを選択
3. パーティが全て追加されます

## 🛠️ 技術スタック
- HTML5
- CSS3（グリッドレイアウト、グラデーション、アニメーション、アコーディオン効果）
- Vanilla JavaScript（フレームワーク不要）
- LocalStorage API
- Blob API（CSVダウンロード）
- FileReader API（CSVアップロード）

## ℹ️ 注意事項
- データはブラウザのローカルストレージに保存されます
- ブラウザのキャッシュクリアやプライベートブラウジングではデータが失われます
- 重要なパーティはCSVダウンロード機能を使用して定期的にバックアップしてください
- CSVファイルは別のパソコンでも読み込み可能です（テキストエディタで編集も可能）
